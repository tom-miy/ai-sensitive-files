# AI Ignore Files

`.aiignore` is a shared statement of intent.
It is not a universal enforcement layer.

Generated files:

- `.aiignore`
- `.cursorignore`
- `.copilotignore`
- `generated/claude-code-deny-read.json`
- `generated/ai-agent-guidance.md`

Different agents and editors support different settings.
Use generated ignore files together with commit checks,
editor settings, and tool-specific deny lists.
Regenerate these files from policy instead of hand-editing them.
